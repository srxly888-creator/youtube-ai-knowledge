## 🔍 使用方法

### 1. 克隆仓库
```bash
git clone https://github.com/srxly888-creator/youtube-ai-knowledge.git
cd youtube-ai-knowledge
```

### 2. 搜索字幕
```bash
# 搜索关键词
grep -r "人工智能" . --include="*.vtt"

# 统计字幕数量
find . -name "*.vtt" | wc -l
```

### 3. 转换为学习笔记
```bash
# 使用 AI 工具总结字幕
python scripts/vtt_to_notes.py
```
