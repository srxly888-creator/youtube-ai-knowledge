## 🎯 使用方法

### 1. 用 Obsidian 打开
```bash
git clone https://github.com/srxly888-creator/youtube-ai-knowledge.git
# 用 Obsidian 打开文件夹
```

### 2. 查看知识图谱
- 打开 `concepts/knowledge-graph.md`
- 切换到阅读模式
- 查看 Mermaid 图谱

### 3. 双链导航
- 点击 `[[概念名]]` 跳转
- 使用 Graph View 可视化
- 查看反向链接

### 4. 字幕溯源
- 打开 `concepts/INDEX.md`
- 查找概念对应的字幕
- 点击字幕文件查看原文
