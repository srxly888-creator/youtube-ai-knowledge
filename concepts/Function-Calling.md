# [[Function Calling]]

## 定义
大语言模型调用外部工具、API、函数的能力，实现从对话到行动的跨越。

---

## 核心要点

### 1. 工作流程
1. 用户请求 → 2. 模型理解 → 3. 决定调用函数 → 4. 执行函数 → 5. 返回结果

### 2. 核心能力
- 参数提取 - 从自然语言提取结构化参数
- 函数选择 - 根据意图选择合适的函数
- 结果解析 - 理解函数返回结果
- 错误处理 - 处理调用失败情况

### 3. 应用场景
- 查询数据库
- 调用 API
- 文件操作
- 发送邮件/消息
